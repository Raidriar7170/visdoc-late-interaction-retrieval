## Validation Before Tag

Use the safe validation profile for release preparation:

```bash
python -m py_compile src/visdoc_retrieve/*.py
python - <<'PY'
import tomllib
from pathlib import Path
tomllib.loads(Path("pyproject.toml").read_text())
print("pyproject parse ok")
PY
pytest -q --ignore=tests/test_final_comparison_run.py
ruff check .
mypy src
openspec validate --all --strict
git diff --check
```

`tests/test_final_comparison_run.py` is intentionally excluded because it
executes final-comparison logic against temporary outputs and reads the final
split. Release preparation must not rerun final comparison or read final labels.
