训练门禁

只有同时满足 allow_real_training=true、 allow_final_test=false、 VISDOC_ENABLE_REAL_TRAINING=1、 本地模型路径存在、CUDA 可用、 max_steps <= 20、 adapter 输出目录被 git ignore，才允许进入 guarded real pilot launcher。

为什么默认不训练

  • allow_real_training_false: Config did not explicitly allow real training.
  • training_env_not_enabled: VISDOC_ENABLE_REAL_TRAINING=1 is required for a real pilot.
  • missing_local_model_path: local_model_path does not exist: local-models/colpali-or-colqwen
  • cuda_not_checked: CUDA was not checked because an earlier non-import gate blocked.

A100 / 本地模型 pilot 运行方式

在 A100 或本地 CUDA 环境中，先准备本地模型目录， 确认 adapter 输出在 ignored artifacts 目录下，再显式设置 VISDOC_ENABLE_REAL_TRAINING=1 并将配置中的 allow_real_training 改为 true。不要下载模型， 不要启用 final test。
